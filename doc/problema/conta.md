# Conta

## Descrição

Representa a conta bancária de uma pessoa.

## Características

* Todos as contas deverão ser persistidas;
* Uma conta está associada à uma pessoa;
* A conta possui o recurso dinheiro.

## Problemas

* Sincronização do recurso dinheiro.
